Stel dat je een examen aflegt waarvoor je een bestand
moet uploaden op x.toledo. Als lector kunnen we
deze inzendingen downloaden als een groot zipbestand.
In dit zipbestand zit per inzending een bestand
waarvan de bestandsnaam volgend patroon vertoont:

    Examen_q0102917_attempt_2017-01-18-10-38-34.txt

Dit bestand bevat informatie over de inzending:

    Name: Anna Roeges (q0102917)
    Assignment: Examen
    Date Submitted: Wednesday, January 18, 2017 10:38:34 AM CET
    Current Grade: Needs Grading

    Submission Field:
    There is no student submission text data for this assignment.

    Comments:
    There are no student comments for this assignment.

    Files:
            Original filename: khhborqc.zip
            Filename: Examen_q0102917_attempt_2017-01-18-10-38-34_khhborqc.zip


Hier zie je dat studente Anna Roeges om 10:38 een zip heeft geüpload.
Dit bestand heeft ze zelf khhborqc.zip genoemd maar
Toledo heeft het hernoemd naar

    Examen_q0102917_attempt_2017-01-18-10-38-34_khhborqc.zip


Begin deze oefening met het uitvoeren van prepare.py.
Dit maakt een subdirectory submissions aan waarin alle .txt bestanden
tezamen met de ingezonden bestanden zitten.

Voor elke student moet je een subdirectory aanmaken
waarvan de naam 'familienaam-voornaam' is, in kleine letters, bv. 'roeges-anna'.
Zorg ervoor dat alle niet-letters uit de directorynaam verwijderd worden:
'Van der Haegen' moet dus 'vanderhaegen' worden.
In deze subdirectory moet je de inhoud van de ingezonden zip extraheren.
