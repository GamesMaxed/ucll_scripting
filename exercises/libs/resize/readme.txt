Voor deze opdracht moet je een script schrijven
dat afbeeldingen/foto's resizet.
Steun hiervoor op Pillow, de Python Imaging Library.

Zoek hoe je Pillow moet installeren (hint: pip).
Zoek vervolgens de documentatie op en implementeer
de gevraagde functionaliteit.

Schrijf een Pythonscript dat indien met het oproept met

    python resize.py <input> <percent> <output>

<input> is een bestaand grafisch bestand (png, jpg, ...).
<percent> stelt een percentage voor, bv. 50
<output> is een grafisch doelbestand met extensie png, jpg, ...

Het script moet <input> openen, het scalen met het gegeven
percentage, en het resultaat wegschrijven naar <output>.


Kopieer enkele grafische bestanden en test je script uit.
